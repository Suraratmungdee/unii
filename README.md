## ติดตั้ง (macOS)
1. Clone repo
```bash
git clone https://github.com/Suraratmungdee/unii.git
cd unii
```

2. เข้าโฟลเดอร์ frontend และติดตั้ง dependencies
```bash
cd frontend
npm install
# หรือใช้ yarn / pnpm ตามที่ต้องการ
# yarn
# pnpm install
```

3. รันในโหมดพัฒนา
```bash
# ถ้าใช้ Vite
npm run dev
